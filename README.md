**AURA** is a low-cost, bio-signal integrated robotic prosthetic designed to bridge the gap between high-end engineering and global accessibility. By leveraging real-time signal processing, this project translates human muscle activity into precise mechanical motion.

---

## 🏆 Recognition
This project was awarded the **Gold Medal in Tech & Engineering** at the **International STEM Olympiad (Frankfurt, 2025)** for its innovative approach to hardware-software integration and social impact.

## 🚀 Key Features
* **Bio-Signal Integration:** Utilizes EMG (Electromyography) sensors to interpret user intent directly from muscle activity.
* **Real-Time Processing:** Optimized Python and C++ logic ensures low-latency motor actuation.
* **Computer Vision Hybrid:** Features an OpenCV-based tracking module for enhanced object interaction.
* **Cost-Effective Design:** Built using accessible components (Arduino, 3D-printed parts) to keep the total cost under $500.

## 🛠 Tech Stack
* **Languages:** Python (Signal Processing, Analysis), C++ (Arduino Firmware).
* **Libraries:** OpenCV (Visual Tracking), NumPy (Data Processing).
* **Hardware:** Arduino, Bio-sensors, Servo Motors.



## 🧬 How It Works
1.  **Signal Acquisition:** Sensors detect micro-voltage changes in the muscles.
2.  **Filtering:** Python scripts apply real-time digital signal processing to remove noise.
3.  **Analysis:** The system identifies movement patterns using custom-built logic.
4.  **Action:** The Arduino controller executes precise PWM signals to drive the robotic motors.
